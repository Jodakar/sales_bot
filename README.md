# 📦 Система автоматизации продаж: Личный кабинет продавца

## 🎯 Общее описание

Система представляет собой **личный инструмент продавца** для управления заказами, товарами и клиентами. Все операции выполняются через VK-бота, который доступен только тебе. Клиенты получают информацию о заказе через мессенджеры (Telegram, WhatsApp) по твоему выбору.

### Ключевые особенности:
- **Бот доступен только продавцу** (одному пользователю)
- **Все заказы оформляются вручную** продавцом на основе разговора с клиентом
- **Клиент получает информацию** о заказе через ссылки на мессенджеры
- **Excel-выгрузки** для аналитики и работы с DeepSeek
- **Синхронизация с 1С и PostgreSQL** для учета товаров и заказов

## 🗄️ Структура базы данных (PostgreSQL)

### Таблица `products` (Товары)
| Поле | Тип | Описание |
|------|-----|----------|
| `id` | SERIAL PRIMARY KEY | ID товара (из 1С) |
| `name` | VARCHAR(255) | Наименование |
| `price` | DECIMAL(12,2) | Цена |
| `stock` | INTEGER | Остаток |
| `description` | TEXT | Описание |
| `category` | VARCHAR(100) | Категория |
| `updated_at` | TIMESTAMP | Дата обновления |

### Таблица `orders` (Заказы)
| Поле | Тип | Описание |
|------|-----|----------|
| `id` | SERIAL PRIMARY KEY | ID заказа (уникальный, автоинкремент) |
| `customer_name` | VARCHAR(100) | Имя клиента |
| `customer_phone` | VARCHAR(20) | Телефон (+7...) |
| `customer_address` | TEXT | Адрес доставки |
| `comment` | TEXT | Комментарий клиента |
| `delivery_method` | VARCHAR(50) | Способ доставки |
| `delivery_time` | TIMESTAMP | Желаемое время доставки |
| `status` | VARCHAR(20) | Статус: `not_paid`, `paid`, `delivered` |
| `total_amount` | DECIMAL(12,2) | Итоговая сумма |
| `created_at` | TIMESTAMP | Дата создания |
| `updated_at` | TIMESTAMP | Дата обновления |

### Таблица `order_items` (Позиции заказа)
| Поле | Тип | Описание |
|------|-----|----------|
| `id` | SERIAL PRIMARY KEY | ID записи |
| `order_id` | INTEGER (FK → orders.id) | ID заказа |
| `product_id` | INTEGER (FK → products.id) | ID товара |
| `product_name` | VARCHAR(255) | Наименование (на момент заказа) |
| `quantity` | INTEGER | Количество |
| `price` | DECIMAL(12,2) | Цена за единицу |
| `total` | DECIMAL(12,2) | Итого по позиции |

### Таблица `customers` (Клиенты — для быстрого доступа)
| Поле | Тип | Описание |
|------|-----|----------|
| `id` | SERIAL PRIMARY KEY | ID клиента |
| `name` | VARCHAR(100) | Имя |
| `phone` | VARCHAR(20) | Телефон |
| `address` | TEXT | Адрес по умолчанию |
| `last_order_date` | TIMESTAMP | Дата последнего заказа |
| `created_at` | TIMESTAMP | Дата первого заказа |
